Retail Demand Forecasting

 Project Overview
This project predicts weekly retail sales using historical sales data and external factors to support inventory planning and supply chain optimization.

Objective
To build a machine learning model that forecasts demand accurately and identifies key drivers of sales trends.

Dataset
Retail sales dataset containing:
- Weekly sales
- Holiday indicators
- Economic factors

Approach
1. Exploratory Data Analysis (EDA)
2. Time-series feature engineering (lag features, rolling mean)
3. Model training (Random Forest, XGBoost)
4. Evaluation using MAE, RMSE, MAPE

Results
XGBoost achieved the best performance with the lowest forecasting error.

Key Insights
- Recent sales trends strongly influence demand
- Holiday periods cause significant spikes

Tech Stack
Python, Pandas, NumPy, Scikit-learn, XGBoost, Matplotlib

How to Run
1. Clone the repo
2. Install requirements
3. Run the notebook

Impact
Helps optimize inventory, reduce stockouts, and improve production planning.
